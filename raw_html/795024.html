      • 天香国色
        Clover举报
        广藿香爱好者以及寻求旧木柜子味道的朋友可考虑入手的一款作品。

        本作的表现随介质和用量多少略有不同：试香纸比皮肤上多了一节近似凉茶的微甜草本的段落；而在皮肤上，用量的多少会影响主体香气的呈现——喜欢木质不太爱奶香的建议近距离喷或厚涂，而喜欢香草的可优先选择远距离薄喷。

        【皮肤上】

        // 用量较多时，广藿香和雪松比较强势。//

        以带油润感的树脂型广藿香开场，混合少量安息香脂和一点微咸的香豆素。3分钟左右，底层有一点黑巧克力式的微苦，雪松亦同时登场并带来粉感木质香，整体质感从油润变为干燥。

        约6分钟时，主体是由广藿香和雪松组成的木质香，味道偏咸，同时基底的间隙开始隐约透出香草的淡淡奶香。10分钟左右，位于周边浓度略低区域的雪松开始带上一点微甜。到15分时，主要香气是带粉尘感的微咸干燥木质（广藿香+雪松），而其中隐约的甜（雪松+香草）则接近纸张，相互结合后组成了老旧木橱+尘封藏书般的画面感。

        20分左右，在高浓度区域贴皮还能闻到一点微苦，而周边浓度较低的地方则有比较明显的香草。同时，手腕外周还围绕着一层能在鼻腔停留的清凉感。

        到40分，高浓度区域的香豆素浓度较高，以致基底都带上了一点类烟熏感。这时中央区域的主体依旧是木质香，而周边香草出没的范围则开始传出安息香脂的气息。后续香气变化较缓慢，香草和香豆素逐渐淡去，最后以安息香脂+木质香结尾。

        // 用量减半之后，香草在中段表现耀眼。//

        用量减少后，香气整体走向不变，但香草的存在感会明显很多。

        从8分钟开始，奶香的香草就占了基底至少三分之一的比重，期间还有点近似玫瑰的花香。香草一直持续到25分左右才开始慢慢淡去，基底恢复尘封微咸木质感，表层则带着安息香脂和轻微玫瑰香气，非常美妙。

        如果你和我一样喜欢Shalimar精粹里那种顺滑的香草，采用远距离薄喷上皮的方式来欣赏这款香水就非常合适了。

        【试香纸上】

        // 有一段皮肤上呈现不出的草本香气。//

        试纸上同样以带树脂感可可型广藿香开场，但没有皮肤上那种油润感，辅助香气是淡淡的安息香脂。香草味登场时间较早，在2分钟左右就能闻到了。

        香气主体在约4分钟左右进入微咸木质的阶段，但整体质感很快就开始向草本靠拢，到10分钟时已变微咸干燥草本，还带轻微的粉感（雪松）。之后，基调逐渐向甘甜靠拢，到12分左右综合香气变为带少许凉感的清甜草本，有那种清热祛暑的广东凉茶的感觉。

        约20分时，基调重新变回微咸，但整体质感依旧以草本为主。这股微咸草本维持了一段时间，到40分左右，香豆素开始冒头，带来略粗糙的微咸质感。大概1小时30分时，综合香气就进展为和皮肤上类似的咸尘封木质感了。

        3小时左右，整体咸度略有下降，一度被压制的香草又重新登场，随后非常缓慢但稳步地增加比重，慢慢挤走木质香……到一天后，剩余咸奶油般的香气结尾。

        ————

        留香中等偏上。皮肤上的扩散度在一开始时只有不到10cm，到10分钟左右开始扩大到20cm。而纸上扩散度则一直都比较低，需要凑近到2cm以内闻。
