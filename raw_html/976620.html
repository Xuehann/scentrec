      • 染香轻醉
        王铁蛋酱举报
        名字直译是芫荽，也就是香菜，有说法是这支实际用的是带有草本气息和柠檬清新感的香菜籽，虽然我也不知道香菜籽到底什么味，本来想买点香菜籽体验一下但是淘宝一斤起售，并没有精致优雅像胡椒一样的小包装遂放弃。总之这玩意儿并不是香菜味。

        按照官方说法是Jacqueline Couturier想用这支香水重新童年记忆里母亲的香水味和格拉斯郊野的气味，而他老公闻了这玩意儿之后夫妻双双把家还，在1972年创立了自己的香水品牌jean couturie.。瓶盖灵感来源于孔雀石，因为制作瓶盖时单独添加色料的关系所以官方号称每个盖子都独一无二，当然这种无聊的事情并不值得当做什么卖点。

        因为有说这个东西停产过之后又调整了配方重新上市，所以曾经我一度以为磨砂瓶子孔雀石花纹盖子的长方瓶子是新版，而parfum de toilette这个富有年代感的老香标志性浓度标识一定是旧版，直到官网打破了我的盲目自信——现在打开他们家官网链接你可以买到从edt到parfum所有浓度，包括parfum de toilette，而这个浓度被他们定义为”更浓缩的edt”而不是常规意识里那个站在edp和香精边界的pdt。如果现在要看包装决定的话我应该会试试那个丑陋的圆柱形版本，似乎那个才是真正的老版本。

        西普玫瑰，直白地说像Sisley缘月，去掉了蜂蜜的浓稠甜蜜，削弱了广藿香的药感，用更清新的橙花、香柠檬和是个老香都爱来一点的醛开头，湿凉的绿意，广藿玫瑰核心与缘月如出一辙，但整体的草本绿意和穿插其中的清新感让这支香水表现得庞大而轻盈，像是更清新草本的缘月，好消息是四季皆宜夏天用也不会很吓人，糟糕的是大冷天这支轻得有点没存在感。

        很便宜，PDT的瓶子实在是庞大而丑陋，作为一个更实用好穿且廉价的Sisley缘月，这支绝对物超所值，除了丑之外唯一的劣势就是留香不太行，基本上大半天散光。

        反正喜欢这一口又不要太重口味的买这个不会有错，非老香爱好者请不要买这个自寻烦恼。

      • 天香国色
        Clover举报
        老版本和新版本一起试了，两者差别非常大，几乎可以看作两款完全不同的香水。

        老版本的香气丰富而浑厚，动物感较强。而新版的香气非常薄弱——尤其是前调和中调，味道既淡扩散又弱，中途还出现了 一些奇怪的味道，只有后调才能勉强算是值得欣赏的香气。

        【老版本】

        // 老式玫瑰动物感西普 //

        蜂蜜型甜玫瑰开场，中间还夹杂着一点铃兰的香气；基底是偏柑橘相的芫荽，以及带着动物感的橡木苔。综合后是质感浓稠的酸甜香气。

        2分钟左右，有微暖的辛辣感浮现，是芫荽的类胡椒面相。约3分钟时，有一阵木质香加入，但持续时间不长。大概5分钟时，酸度几乎完全消失了，剩下蜂蜜般的甜玫瑰和动物感橡木苔。

        到10分钟左右，玫瑰香气中掺入了一些带青绿感的微酸老鹳草。15分左右，原本模糊的动物感转变为比较清晰的麝猫香，带轻微的动物型皂感。

        约22分时，主体香气由玫瑰、橡木苔、柑橘木质和麝猫香复合而成。同时，在高浓度区，麝猫香和芫荽带来的类胡椒香气混合出了微脏的动物感。这一阵香气非常稳定，直到8小时还是这个味道。

        到10小时左右，玫瑰花香淡去，剩余轻微动物皂感的麝猫香+橡木苔，凑近还能闻到非常微弱的药感广藿香。这一段落有点接近毕加索同名的后调，但毕加索同名的皂感比本作更为强烈。

        【现行版本】

        // 基本上只有后调好闻，前中调气味很割裂 //

        开场的玫瑰花香非常霸道，蜂蜜般的甜玫瑰香气盖过了其他的气味。直到3分钟左右，甜度减淡，基底透出一点干燥的果皮感。

        约8分钟时主体变得非常酸：高浓度区域呈现出水果接近果核区的那种酸涩感，而低浓度区域除了酸感以外，还有一点当归混天竺葵的香气。

        12分左右，高浓度区出现了一种奇特的烧焦塑料味，闻起来就像因为短路而烧焦融化的电线外层塑料皮……总之不是一种让人愉悦的味道。到18分左右，这股“烤胶”味散去，主体恢复为当归+玫瑰，低浓度区则恢复为蜂蜜型玫瑰。可惜，到28分时，烤塑料味在高浓度区又卷土重来了……如此反复。

        一直到4小时左右，远闻开始有一丝药感广藿香的气息，这时候基底主体仍然是玫瑰，但整体香气总算是朝着后调的方向前进了。

        到4个半小时，玫瑰香气基本散去，而广藿香已经下沉到基底。这时候的广藿香还不是太突出，基调以橡木苔，以及芫荽/香根草组成的酸感木质为主。这个版本的橡木苔没有那种旧式的动物感，只带着非常轻微的皂感型麝猫香。

        之后一直到10小时，香气变化非常缓慢：广藿香占比增大，带木质感+中等程度药感；橡木苔减淡；芫荽只剩下微弱酸感；而微皂感麝猫香依旧坚挺。综合起来的气味还是挺好闻的，也幸亏有这么正常的后调才帮新版本勉强拉回了一点分数。

        ————

        不论新版还是旧版，留香都非常持久，在手腕上有12小时以上。至于扩香范围，差异就非常大了——老版本中等偏上，而新版本范围较小。
