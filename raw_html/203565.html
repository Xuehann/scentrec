      • 天香国色
        古月荒原举报
        正装很难买的一款香。

        颜色很好看，液体质地黏厚，如同药酒，难以想象下了多少猛料。

        上皮，没有如诸位香友所形容的充满欲望的味道，和性挂不上什么边，但确实是脾气蛮暴躁的一支香，前调非常有攻击性，极其强势的广藿香和艾蒿开篇，脏感皮革，阴暗且厚重，真的完完全全就是药酒的味道，但又没有任何一种药酒能如此炸裂如此惊艳，如一个穿着老皮衣的英伦绅士却开了个中药铺子的味道。

        中后调逐渐缓和下来，带着晦暗的木质基底，香草和檀木带来清甜，不凋花的幽香成为主角，此花香不轻盈，极具质感，细腻绵长，温润沉厚。后调真的好暖好甜，丝毫不腻的那种甜，自然散发不矫揉造作，像阳光暴晒过的被子的那种甜。

        前后调的反差，是我目前用过的所有香里最大的，前调有多暴躁有多撕裂，后调就有温柔，也正因如此它才迷人。

        复杂的香料成分使得其带着一丝老香的神韵，富有年代感，气势恢宏，大反派的味道，不过我确实无法把这支香和性什么的联系起来，虽然味道越闻越令我上头。再过五年到30岁了估计我才能勉强驾驭这支香，大多数人还是会难以接受，另类的香水，明白香水未必是香的是每一个入坑人的必经之路。

        由于个性太足，不要在任何日常场合穿他，务必三思，尤其是上班期间，会扰民哦，私下里就不一样了，经常侍寝，个人最爱的几款之一。

        扩散极强，留香极长，黑洞体表示很满意，截止目前留香扩散都极为理想的一款。

        1740这支香的后调总能唤起我对故乡怀念，童年时代夏日闷热的夜晚，被蚊子折磨的半梦半醒的我，感知到妈妈撒了点花露水在我四周，点起一盘蚊香然后把风扇打开，使我能再度安然如梦。早上起床，屋子里残留的蚊香，晨曦洒落，而妈妈在喊你吃早餐。

        希望大家不要被各路神乎其神的评价吓到，真的是很温柔的一支香。

      • 香界传奇
        🌋白唯🌄举报
        重口萨德侯爵，我的香评终于来了。之前实在是too busy，以至于现在还有几百个款没嘚时间写，言归正传了。

        入鼻的一瞬，这重口个毛啊，甜亮而湿润的类柑苔气息首当其冲，随后湿漉的广藿香将其托出后，被包裹的辛辣元素（小豆蔻）与苍烈木质气息（不凋花与树脂）张牙舞爪般从中挣扎而出。

        随后沙哑的葛缕子还带着辛辣与臭味，投向了冰与火的大海之中且增添了几许咸汗味。

        华丽而稳重，却不丧失任何质感与细节，这是一款以经验而论我闻到30秒就知道必然好的作品。清新通透的广藿艾蒿与橡苔等辛凉元素无缝衔接，将整个通透彻净的凉爽汤底平铺于人眼前。但本作绝非是什么清新凉爽之辈，而是在这份凉感之上，不断分离与变幻出各类样式的辛辣暖热气息。这种手笔的确是会让人感受到“冰火交融”的感觉，也是所谓肉欲系香水的一个象征。

        说实话的，我个人完全没有联想到什么：黑色、鬼魅、皮鞭、sm、脏与重口等等这类词汇。

        又或许是我体验过太多重口的东西，在老香馥奇中这类东西频频可见，随便举个例子譬如华氏温度与卡郎的亚特汗。而所谓的“皮革腥臭”来说，这份骚感皮革与沙哑气如果跟毕坚同名相比也是小巫见大巫了，我觉得在我看来，这家伙就像是老香新作的致敬感（异曲同工的似乎还有尼古莱）。我个人认为这里的皮革来自于树脂类，还是属于能接受的那种。整体的尘土气息、药感，以及熟悉的香草融入劳丹脂之结合，这都是老香的经典手笔了。

        留香与扩散上乘，我真心觉得一点都不下贱，或许有些肉欲与骚感，这我承认，但绝不低级。

        随着时间的推移，木质的核心轮廓逐渐清晰，馥奇的元素也各自展开。整个体验还是秉承者冷热交替的矛盾感与扩展度，元素的运用比例很微妙，质感也上乘。

        香水故事的确是精品频出，奈何赏识者还是不多。

        上皮：上皮后整个质感与层次更加清晰了，首当其冲的的确是柑苔类熟悉的柑橘气息与凉爽草本+湿感（就是艾蒿）。随后皂感闪过，与广藿交织者洁净感，并且带出了辛辣刺激的豆蔻芫荽与尘感，以及动物气息。随着时间的推移，整个气味的力度加大，一开始那般清新脱俗的雅致感已然变成了一副中年大叔留着胡茬抽着雪茄的既视感，在此时清亮的柑橘甜感仍然是可以寻觅得到的，另外还加入了榄香脂增加油润度。

        但请注意，绝非油腻！这款香仍是以辛辣刺激感为领头，葛缕子带来的气息愈发沙哑，且更加增添汗味。桦木焦油伴随着琥珀以及之前的动物+尘土质感，成为了一个独到的皮革香气，我的感受还是比较像淡化版的毕坚，又有几分相像于华氏温度、卡地亚飞行，还有tf的一款等等。

        即便如此，艾蒿还是遥相呼应。

        总之骚是骚，但在我身体上体现出来的味道其实并没有那么骚，反而是粗犷的男性特征。我觉得它配范思哲eros的那个宣传画是正好不过了。

        对了，这家伙的后调特别细腻温润，可真是个完整的故事。

        大约5分钟时酸感退减，同时桦木的烟熏感出来了，配合微苦的木质广藿香，是一段微焦苦烟熏木质，算是这个组合里面最苦的一个段落了。接下来几分钟，即使木质广藿香一度盖过了桦木，但有劳丹脂的中和（加上我的高温皮“烘烤”），综合出来的香气并不苦。

        10分钟左右，离远闻是微酸的不凋花+甜树脂+木质香+小豆蔻（没错，这里的小豆蔻很霸道）。近闻时香气种类反而不多：低浓度区则是酸甜的不凋花+劳丹脂，而高浓度区是干燥木质（雪松+广藿香）+小豆蔻，

        约20分时，不凋花加入到高浓度区域，主体香气变为略潮湿的酸甜木质；而低浓度区除了增添了清香的榄香脂以外，表层还多出一阵动物感，不臭，但家里有养毛孩子的人应该会觉得有点熟悉。

        30分，远端的环境香气里除了树脂香气以外还有一阵轻微的皮革味，接近麂皮衣饰的气味。近闻的香气变化不大，依旧是酸甜木质，大致比例为潮湿木质5:微甜树脂2:不凋花1。

        40分左右，近闻时的香气竟然带给我一种冬日在壁炉旁烤火的惬意画面感：微暖的木质（柴火）、香草甜酒，还有酸甜的糕点……

        在1小时左右，木质感降低，主体香气变为甜度近似蜂蜜的稀树脂质感，成分以榄香脂为主，香草其次，不凋花第三。直到1小时10分，木质感又重新加强，甜度也开始下降，香气慢慢向酸甜树脂木质靠拢并持续到最后。大概8小时的结尾阶段甜度和木质完全散去，剩下微酸树脂。

        小结：只在这一组合出现的香气是小豆蔻、芫荽、动物感。

        【试香纸】

        // 先苦后酸甜，可辨别的香气种类略少，但仍有一定的变化和层次感。//

        开场是有树脂包裹感的微苦广藿香，香气近似黑巧克力。4分钟左右树脂感减退，质感变为干燥的烟草，微苦，没有常见的甘甜气息。

        10分，木质感开始慢慢盖过草质感。15分左右，微树脂感开始浮现。到20分时，基底已经转为干燥的木质（雪松），而香气的主角则变成了清甜的榄香脂。

        30分，不凋花出场，主体香气变为酸甜树脂+微潮湿木质+清凉感。40分左右，不凋花的酸度减弱了一些，带轻微烟熏味的桦木加入，烟草感又回来了一些。

        1小时左右，香气以微苦的木质为主，但时不时会有微酸的不凋花香气闪现。1小时15分，树脂感增强，主体恢复为酸甜树脂+木质香。

        约1小时40分时，突然浮现出一阵较强的皂感，一度盖住了其他香气，整体变成了草木型香皂感。到1小时45分左右，皂感退去，恢复酸甜树脂木质，之后气味基本稳定下来直到最后。

        【皮肤 + 低浓度】

        // 相对最无趣的一个组合，香气种类少、层次少、变化少。//

        开场就是微苦木质广藿香，基本没有其他的辅助香气。直到15分时，带微烟熏味的桦木成为主角。到30分左右，主体转变为酸甜的不凋花+木质香基底，持续了非常长的时间。
